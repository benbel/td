## TD d'économie publique

Ce dépôt git regroupe une partie des fichiers utilisés pour les conférences
de méthode d'économie publique données à Sciences Po au semestre d'automne 2018
(cours de Camille Hémet) et au semestre d'automne 2019 (cours de Pierre-Hernan
Rojas).

Il est stocké ici : `https://github.com/benbel/td`
